What it does

📊

10+ Metrics

Extract efficiency, planning ratio, refactoring rate, bug density, and more from every session.

🧬

Builder Archetypes

Are you an Architect, Sprinter, Debugger, Collaborator, or Autonomous Agent? Find out with confidence scores.

📈

Trend Analysis

Track how your coding patterns evolve across sessions. Spot improvements and regressions.

📄

Rich Reports

Generate beautiful terminal dashboards and shareable HTML reports with charts and insights.

🔒

100% Local

Zero network calls. Your code never leaves your machine. Read-only, never modifies source files.

⚡

Zero Config

Works out of the box with Claude Code's JSONL format. No setup, no accounts, no API keys.

Ways to use it

Pick the flow that fits — every option keeps your data on your machine.

Inside Claude Code

🧩 The /ai-fluency skill

Run it as a Claude Code skill — Claude Code itself analyzes your sessions. No extra model, no API key, no install beyond the repo.

/ai-fluency
/ai-fluency --dir ~/.claude/projects
Standalone + local AI

🤖 CLI with a local model

Run the CLI anywhere. A small local model via Ollama (default gemma3:4b) writes your profile — all on-device.

ollama pull gemma3:4b
claude-insight --dir ~/.claude/projects
No model needed

⚡ Heuristic / offline

Skip the AI entirely for instant, deterministic analysis — handy in CI or when Ollama isn't installed.

claude-insight --no-ai
claude-insight --report report.html
For tooling

🔌 JSON export

Emit the raw metrics and a sample of prompts as JSON to pipe into your own scripts, dashboards, or models.

claude-insight --json --dir ~/.claude/projects

How it works

1

Parse

Reads Claude Code JSONL transcripts from ~/.claude/projects/ — every message, tool call, and file edit.

2

Analyze

Extracts metrics like token efficiency, planning ratio, and iteration speed using local computation.

3

Profile

Assigns builder archetypes based on your patterns — with confidence scores and trait breakdowns.

4

Report

Generates terminal dashboards or HTML reports with visualizations you can save and share.
